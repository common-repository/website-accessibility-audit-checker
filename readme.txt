== Description ==

[AAArdvark](https://aaardvarkaccessibility.com/?utm_source=website&utm_medium=promo-link&utm_campaign=wordpress-org) is a service that scans and continuously monitors your site for WCAG accessibility issues. This plugin provides access to accessibility reports directly in your WordPress administrative dashboard.

This plugin connects to the AAArdvark Accessibility API (https://api.aaardvarkaccessibility.com) to deliver reports directly in your WordPress dashboard. Requires an account on our platform [app.aaardvarkaccessibility.com](https://app.aaardvarkaccessibility.com/?utm_source=website&utm_medium=promo-link&utm_campaign=wordpress-org).

If you experience any issues with this plugin or need help setting it up, please contact our [support team](https://aaardvarkaccessibility.com/contact-us/?utm_source=website&utm_medium=promo-link&utm_campaign=wordpress-org).

[See Terms of Use for AAArdvark](https://aaardvarkaccessibility.com/terms-of-service/?utm_source=website&utm_medium=promo-link&utm_campaign=wordpress-org) and [Privacy Policy](https://aaardvarkaccessibility.com/privacy-policy/?utm_source=website&utm_medium=promo-link&utm_campaign=wordpress-org).

== Installation ==

1. Upload `AAArdvark` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit `Settings` > `AAArdvark` in the administrative dashboard.
4. Enter the API Key you created for this site from your [app.aaardvarkaccessibility.com](https://app.aaardvarkaccessibility.com/?utm_source=website&utm_medium=promo-link&utm_campaign=wordpress-org) account, and select the user roles that you would like to have access to the reports page.
5. Results from your latest accessibility scan will now be available at `Tools` > `Accessibility Report` in the administrative dashboard.

= Where do I find my API Key? = =

* From your site dashboard on [app.aaardvarkaccessibility.com](https://app.aaardvarkaccessibility.com/?utm_source=website&utm_medium=promo-link&utm_campaign=wordpress-org), go to the `Integrations` menu item and click "Add Integration".
* Select the `CMS Plugin` integration from the list.
* The API Key will be displayed on the next screen. Make note of it as you will not be able to view it again.

= What do I do if I've lost my API Key? = =

You can generate a new API key from your site dashboard on app.aaardvarkaccessibility.com.

* From your site dashboard on [app.aaardvarkaccessibility.com](https://app.aaardvarkaccessibility.com/?utm_source=website&utm_medium=promo-link&utm_campaign=wordpress-org), go to the `Integrations` menu item, find the `CMS Plugin` integration and click `Configure`.
* Click the `Regenerate Key` button and confirm.
* The new API Key will be displayed and your old API Key will be disabled.

= Where did the name "AAArdvark" come from? = =
The name for AAArdvark came from the animal called an aardvark! Since the word already starts with two A’s, we thought it would be fun to add another “A” as a reference to WCAG’s highest level of accessibility compliance.
